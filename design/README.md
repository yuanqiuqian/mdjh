# 设计文件目录

本目录仅包含设计与规格文件，用于在正式开发前对玩法、数据、AI叙事、PWA体验等关键问题达成一致。

文件说明：

- overview.md：产品目标、核心体验与范围边界
- systems.md：角色属性、门派、战斗、好感度、物品与存档等系统设计
- ai_context.md：AI剧情/对话生成、角色生成、上下文记录与动态压缩方案、提示词模板
- pwa_ux.md：PWA目标、信息架构、页面流程、离线与存档策略、隐私与安全
- open_questions.md：待确认问题清单与建议默认值

当前已落地的重要设计更新：

- 对话模式 / 交战模式双模式结构
- 交战模式采用本地回合制，不逐招请求AI
- AI负责战斗切入判断、战斗初始化与战后剧情续写
- 战利品、金钱与关系变化在战斗结束后真实落盘
